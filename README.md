# Java Projekt für AD Exercises

## Gelöste Aufgaben
* **SW00** - Demo Klasse
* **SW01** - Übung: AhaBeispiel aus den Folien, Fibonacci, SymetrieDemo
* **SW02** - Übung: Arraysringbuffer, ArrayStack List, Queue
* **SW03** - Übung: Bäume, Binärbäume und Suchbäume
